Для рассылки сообщений на всех пользователях единовременно, не превышая ограничения Телеграм API по скорости, можно использовать следующие способы:

1. Использовать очередь задач: Создать очередь задач, в которую добавляются все сообщения для отправки пользователям. Затем использовать несколько воркеров, которые будут извлекать задачи из очереди и отправлять сообщения пользователям.

2. Использовать распределенную обработку: Разделить все сообщения на несколько частей и передать каждую часть на обработку отдельной инстанции бота. Таким образом, каждая инстанция будет отправлять сообщения своей группе пользователей параллельно другим инстанциям.

3. Использовать сторонние сервисы.
